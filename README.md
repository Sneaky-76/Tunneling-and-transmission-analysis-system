# Tunneling-and-transmission-analysis-system
Repository for a project combining subjects Network Programming with Telecommunication Networks and Systems, conducted in the year 2025/26.
The goal of the project is to make a functional tunneling system with the analysis of its transmission parameters, consisting of both client
and serwer applications, written in C++ language.
